The GeoPlatform Account Request Tool provisions EPA GeoPlatform user accounts during emergency response events. Allows authorized users (Field Team Coordinators) to rapdily set up users for field data collection events and provides access to specific groups.

Steps:
1. Become a Field Team Coordinator
2. Configure a new response
3. Request new GPO accounts
