The GeoPlatform Account Request Tool provisions EPA GeoPlatform (GPO) user accounts during emergency response events. Authorized users (Field Team Coordinators) can rapidly set up accounts for field data collection events and provide access to specific GeoPlatform groups. Visit the Help Documentation for additional information.

Steps:
1. Become a Field Team Coordinator
2. Configure a new response
3. Request new GPO accounts
