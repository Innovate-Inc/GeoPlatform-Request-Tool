The GeoPlatform Account Request Tool provisions EPA GeoPlatform (GPO) user accounts during emergency response events and projects. Authorized users (Team Coordinators) can rapidly set up accounts for field data collection events and provide access to specific GeoPlatform groups. Visit the Help Documentation for additional information.

How to get started:
1. Become a Team Coordinator (and assign Delegates)
2. Configure a New Response or Project
3. Request New GeoPlatform User Accounts
